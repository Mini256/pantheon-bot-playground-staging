# Mini256-pantheon-bot-dev-playground

## Hello World page

This repository includes a small, modern HTML5 "Hello World" page (`index.html`)
with styling in `styles.css`.

### View the page

1. Open `index.html` directly in your browser, or
2. Serve the folder locally (recommended):

```bash
python3 -m http.server 8000
```

Then visit `http://localhost:8000/`.
