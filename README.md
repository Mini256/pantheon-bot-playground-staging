# Mini256-pantheon-bot-dev-playground

## Hello World Page

This repository includes a simple, modern, responsive static page:

- `index.html`
- `styles.css`

### View The Page

Option 1: open directly in a browser

- Open `index.html` with your browser of choice.

Option 2 (recommended): serve locally over HTTP

```bash
cd pantheon-bot-dev-playground
python3 -m http.server 8000
```

Then visit `http://localhost:8000` and you should see "Hello World".
